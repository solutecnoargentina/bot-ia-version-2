# SOLUTECNO SAAS

Sistema SaaS autogestionado para bots de WhatsApp con IA local (Ollama + Qwen)

## Estructura

- backend → servidor Node.js
- frontend → dashboard
- sessions → sesiones de WhatsApp
- logs → registros del sistema

## Objetivo

Plataforma profesional para:
- conectar WhatsApp
- gestionar agentes IA
- responder mensajes automáticamente
- escalar a múltiples clientes

## Estado

Proyecto iniciado desde cero
